Peralta Informatics is a software development workshop — we build high-quality tools and web applications. The focus of the company is on solid, scalable architecture, interface usability, and product correctness.

We have significant expertise across the technology spectrum, with particular emphasis on XHTML/CSS/AJAX techonologies, CLR (.NET/Mono) with C#, ASP.NET/XSP (C#), Python/mod_python application development, and server-side XML technologies (XML Web Services, XML RESTful services, XSLT, XQuery, and Relax NG).

We like Unix and have a deep knowledge of the GNU/Linux and FreeBSD operating systems.
